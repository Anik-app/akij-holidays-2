☁️ GitHub Cloud Storage

Configure once. Every document you save is automatically pushed to the GitHub repository, and every time you open the app your documents sync back. Credentials are stored in this browser only.
Checking…
How to set up:
  1. Create a repository (e.g. akij-holidays) on GitHub.
  2. Generate a fine-grained personal access token at github.com/settings/personal-access-tokens/new with access limited to that repo and Contents: Read and write.
  3. Paste the values below and click Save & Test.

🔒 For security, the token is not hardcoded. Paste it once; it will be stored safely in your browser.
